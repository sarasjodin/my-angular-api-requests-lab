About this site

This site is part of a frontend development lab focused on Angular 20, RxJS, and working with external data sources.

The application fetches a course schedule from an external API. If the remote source is unavailable, it automatically falls back to a local JSON file stored in the public folder. Course names are translated via the local JSON to improve accessibility for international users.

It demonstrates several core Angular concepts including:

  • Standalone components
  • HTTP requests using HttpClient with RxJS
  • Data binding with classic Angular techniques
  • Filtering and sorting in real-time
  • Responsive table layout with accessibility in mind

The table view allows users to search for courses by name or code, sort by any column (ascending, descending, or reset), and view syllabus links for each course.

The project is fully version-controlled with Git, hosted on GitHub, and published on a public web host. Styling is done using SCSS at both component and global levels.
